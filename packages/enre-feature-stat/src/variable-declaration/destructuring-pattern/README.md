# Destructuring Pattern in Variable Declaration

## Patterns

```js
//    Not a destructuring pattern
//    v
const a = 1;

//    An object destructuring pattern
//    vvvvvv
const {a, b} = obj;

//    An array destructuring pattern
//    vvvvvv
const [a, b] = arr;
```
